# 波次里程碑与阶段总验收矩阵

## 1. 波次里程碑

| 波次 | 包含 Step | 里程碑定义 |
| --- | --- | --- |
| `Wave A` | `00-05` | 共享主干冻结，可放开引擎并行 |
| `Wave B` | `06-09` | 三大引擎与动作层主链路闭环 |
| `Wave C` | `10-12` | 插件、交付、性能治理闭环 |
| `Wave D` | `13` | 全链路发布门禁完成 |

## 2. 波次总验收

| 波次 | 通过标准 |
| --- | --- |
| `Wave A` | 项目、资产、AI、桌面权限主干全部冻结 |
| `Wave B` | Film / MagicCut / Canvas / Chat-Prompt 至少各有一条主链路可跑通 |
| `Wave C` | 插件、发布、性能预算均有可执行证据 |
| `Wave D` | E2E、构建、制品、Go/No-Go 结论完整 |
